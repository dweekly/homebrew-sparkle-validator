# Homebrew Tap for Sparkle Validator

This is the official Homebrew tap for [Sparkle Validator](https://github.com/dweekly/Sparkle-Validator).

## Installation

```bash
brew tap dweekly/sparkle-validator
brew install sparkle-validator
```

## Usage

```bash
sparkle-validator appcast.xml
sparkle-validator https://example.com/appcast.xml
```
